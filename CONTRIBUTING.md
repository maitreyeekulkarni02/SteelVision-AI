# Contributing

Thank you for your interest in contributing to SteelVision AI.

## Getting Started

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes.
5. Commit using clear commit messages.
6. Open a Pull Request.

## Coding Standards

- Follow PEP 8.
- Keep functions modular.
- Add comments where necessary.
- Write meaningful commit messages.

## Reporting Issues

Please use GitHub Issues to report bugs or suggest enhancements.

Thank you for helping improve SteelVision AI.
