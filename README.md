# 🚀 Overview

SteelVision AI is an affordable **Edge AI powered industrial inspection platform** designed to help manufacturers detect machine defects, monitor machine health, and generate predictive maintenance recommendations.

The system uses **Computer Vision and AI-based defect intelligence** to reduce unexpected machine failures, improve inspection efficiency, and support Industry 4.0 transformation.

---

# 🎯 Problem Statement

Manufacturing industries face several challenges:

- Unexpected machine breakdowns
- High maintenance costs
- Manual inspection limitations
- Delayed defect identification
- Production downtime


Traditional inspection methods are:

- Time consuming
- Human dependent
- Difficult to scale

# ✨ Key Features


## 🔍 AI Computer Vision Inspection

- Machine image analysis
- YOLO based object detection
- Defect identification
- Confidence estimation


## 🏭 Industrial Defect Intelligence

The platform supports detection of:


- Rust
- Crack
- Oil Leakage
- Corrosion
- Loose Bolt
- Damaged Belt
- Missing Component
- Surface Damage


---

## 📊 Machine Health Monitoring

Generates:

- Machine Health Score
- Condition Status
- Maintenance Priority
- Risk Analysis


Example:
Health Score: 85%

Status:
Good

Maintenance Priority:
Medium


---

## 🔧 Predictive Maintenance Recommendation


AI provides maintenance guidance based on detected defects.

Example:
Detected:
Rust + Surface Damage

Recommendation:
Schedule preventive maintenance
and inspect affected components.

## 📄 Digital Inspection Reports

Generate downloadable reports containing:

- Inspection date/time
- Detected defects
- Confidence score
- Machine health
- Maintenance recommendation
