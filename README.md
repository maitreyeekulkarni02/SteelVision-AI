# 🏭 SteelVision AI

## Edge AI Powered Industrial Inspection Platform

SteelVision AI is an Industry 4.0 industrial inspection platform that uses Computer Vision and Edge AI to detect machine defects, estimate machine health, prioritize maintenance, and generate digital inspection reports.

Developed as part of the Tata Technologies Innovent Hackathon.

---

# Features

## AI Inspection

- AI-powered defect detection
- Industrial image inspection
- Live camera inspection
- Real-time video inspection
- Bounding box visualization
- Confidence score analysis

---

## Supported Defects

- Rust
- Crack
- Oil Leakage
- Loose Bolt
- Corrosion
- Damaged Belt
- Missing Component
- Surface Damage

## Machine Intelligence

- Machine Status Classification

- Defect Severity Analysis

## Dashboard
